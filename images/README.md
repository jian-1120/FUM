# FUM Visual Assets

This directory is reserved for professional presentation assets used in README previews, GitHub releases, and future Blender Market materials.

| Directory | Purpose |
|---|---|
| `banners/` | Repository headers and marketplace banner images. |
| `screenshots/` | Static screenshots of the FUM panel and inspection results. |
| `gifs/` | Short workflow demonstrations for Export Preflight and individual checks. |

Only polished, final presentation assets should be committed here. Temporary captures, drafts, and generated test files should remain outside the repository.
