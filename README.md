# FUM — Fix Your Mesh

**FUM 是一款专业级的 Blender 网格验证管线工具。**

FUM 旨在帮助技术美术师、3D 资产作者和管线团队，在将模型导出到游戏引擎、渲染农场或任何下游生产工具之前，确保其几何体的完整性和导出就绪性。它将繁琐的网格检查流程转化为一个高效、可靠的验证步骤，是您 3D 生产工作流中不可或缺的一环。

## 🎯 目标用户

*   **技术美术师：** 确保资产符合技术规范，减少返工。
*   **3D 资产作者：** 验证模型质量，提升交付标准。
*   **管线团队：** 统一团队的网格质量标准，优化生产流程。
*   **游戏资产创作者：** 确保模型在游戏引擎中表现良好，避免运行时问题。
*   **3D 打印创作者：** 检查模型是否适合 3D 打印，避免物理缺陷。

## 🚀 核心功能：导出预检工作流

FUM 的核心优势在于其独特的 **导出预检工作流**。它超越了简单的网格检查，提供了一个统一的验证系统，能够快速评估活动网格的导出就绪性，并以清晰的 **PASS**、**WARNING** 或 **FAIL** 状态进行总结。

此工作流显著提升了生产可靠性，通过在早期阶段发现并突出显示潜在问题，从而避免了后期代价高昂的修复和项目延误。

| 状态 | 含义 | 建议操作 |
|---|---|---|
| **PASS** | 未检测到已知网格问题。 | 网格已准备好导出。 |
| **WARNING** | 检测到非阻塞性问题。 | 导出前请检查高亮显示的几何体。 |
| **FAIL** | 检测到阻塞性导出问题。 | 导出前请修复报告的问题。 |

> FUM 不会自动修改您的网格。它检测、高亮显示并总结潜在问题，以便艺术家能够控制最终的清理过程。

## 💡 FUM 与 Blender 原生工具的对比

Blender 原生工具提供了基础的网格选择功能，但它们通常是：

*   **碎片化的：** 需要在多个菜单和模式之间切换。
*   **手动的：** 依赖用户手动触发和解释结果。
*   **非工作流导向的：** 缺乏一个统一的、以导出为中心的验证流程。

相比之下，FUM 提供：

*   **统一的验证工作流：** 一键运行所有关键检查。
*   **PASS/WARNING/FAIL 系统：** 清晰直观的导出就绪性反馈。
*   **导出导向的检查：** 专注于生产环境中常见的网格问题。

FUM 将这些独立的检查整合到一个高效的管线工具中，显著提升了工作效率和模型质量。

## ⬇️ 安装指南

1.  从 [GitHub Releases 页面](https://github.com/jian-1120/FUM/releases/latest) 下载最新版本的 `FUM_vX.X.zip` 文件。
2.  在 Blender 中，前往 `编辑 > 偏好设置 > 插件`。
3.  点击 `安装...` 按钮，选择您下载的 `FUM_vX.X.zip` 文件。
4.  启用 **FUM - Fix Your Mesh** 插件。
5.  在 3D 视口中，按下 `N` 键打开侧边栏，即可找到 **FUM** 面板。
