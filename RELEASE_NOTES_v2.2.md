# FUM v2.2 — Commercial Packaging Update

This update focuses on transforming FUM into a trustworthy, commercial-grade Blender pipeline tool, emphasizing professional presentation, repository cleanliness, product positioning, and distribution readiness.

## Product Presentation

*   **README Productization:** The `README.md` has been completely rewritten into a professional English product landing page, focusing on FUM as a pipeline-oriented, export validation tool.
*   **Target Users:** Added a dedicated section to clearly define the target audience, including Technical Artists, 3D Asset Authors, Pipeline Teams, Game Asset Creators, and 3D Printing Creators.
*   **Workflow Focus:** Highlighted the Export Preflight workflow as the core selling point, explaining its benefits for production reliability and comparing it with fragmented native Blender tools.
*   **Visual Asset Preparation:** Prepared the repository structure and README sections for banners, demo GIFs, and multiple feature screenshots.
*   **Future Direction:** Simplified the roadmap to focus solely on FUM Pro for Blender, avoiding overpromising on other platforms.

## Branding & Licensing

*   **Branding Cleanup:** Updated author wording to "Jian – 3D tools developer" across all relevant files, removing any non-professional or self-evaluative language.
*   **License Update:** Changed the project license from MIT to GPLv3 to protect open-source distribution and reduce the risk of closed-source commercial reselling. All license references have been consistently updated.

## Repository Hygiene

*   **Permanent Internal File Removal:** Ensured complete deletion of `PROJECT_STATE.md` and `docs/ENGINEERING_NOTES.md`.
*   **.gitignore Reinforcement:** Updated `.gitignore` to permanently block `PROJECT_STATE.md`, `PRODUCT_GOVERNANCE*.md`, and `ENGINEERING_NOTES*.md` from appearing in the repository.
*   **Release ZIP Verification:** Verified that the GitHub Release ZIP is properly installable with the correct Blender addon structure, avoiding repository-root nesting.

## Version Information

*   **Version Target:** v2.2 "Commercial Packaging Update"
*   **Blender Target:** Blender 4.x
